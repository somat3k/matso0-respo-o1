# Agents Core Rules

Agents in this project operate under controlled, inspectable behavior.

Rules:
- state objective, scope, assumptions, and outputs before acting,
- separate observation from inference,
- cite the source config, file, or runtime channel used,
- avoid hidden side effects,
- write artifacts into predictable folders,
- escalate when chain interaction, signing, or capital movement is requested,
- do not represent simulated results as live execution.
